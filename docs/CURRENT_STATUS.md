## محدودیت‌های شناخته‌شده

1. **Long Polling:** نیاز به پروسه دائمی دارد — serverless deploy ممکن نیست
2. **کد همکار مشترک:** یک کد برای همه — امنیت پایین
3. **رسید پرداخت:** `file_id` Bale ذخیره می‌شود — بدون آرشیو محلی
4. **tmp/ directory:** در runtime ساخته می‌شود برای PDF — ممکن است در برخی PaaS‌ها مشکل ایجاد کند
5. **وضعیت مکالمه در DB:** هر پیام یک query DB می‌زند برای خواندن/نوشتن state
